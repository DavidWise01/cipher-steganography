# STEGANOGRAPHY · STG

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **HERMES**.

STEGANOGRAPHY (STG) — Paper I of VI in the AUDIO CIPHER CHANNELS series: cryptography hides a message's MEANING, steganography hides its EXISTENCE (στεγανός+γραφή, 'covered writing'). The lineage (Herodotus' tattooed scalp ~440 BC → Trithemius coins it 1499 → WWII microdots → Simmons' 1983 prisoners' problem → digital LSB/watermark), the FIVE audio methods (LSB · echo-hiding · phase · spread-spectrum · psychoacoustic), and the capacity–imperceptibility–robustness TRIANGLE (pick two). Live: a WORKING echo-hiding watermark — 8 bits embedded as echo delays (short=0/long=1), the marked carrier sounds identical, recovered by autocorrelation (verified 40/40 random trials). ⚑ Watermarking is steganography pointed at provenance, but it proves a mark is PRESENT, not that you made it — the fix is a signed content-hash anchored externally (the ROOT0 witness move). Two-layer honest: lineage + methods are textbook; the demo hides in clean synthetic noise, real attacks (recompression, StirMark) are harsher. In the comms layer with [[pulse]]; sibling to [[cipher-ultrasonic]].
